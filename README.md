# 歌词学习助手

帮助用户学习英文歌曲的 Chrome 浏览器扩展。通过控制歌词逐句重复播放次数，让用户能够充分理解每句歌词的发音和含义。

## 功能特性

- **重复次数设置**：自由设置每句歌词重复播放次数（1-10 遍）
- **自动循环播放**：计数未达标时自动回跳，达标后自动跳转下一句
- **快捷键操作**：支持键盘快捷键控制播放
- **实时状态显示**：Popup 界面显示当前歌词和学习进度

## 安装使用

### 环境要求

- Chrome 浏览器（版本 88+）
- Node.js（用于编译 TypeScript）

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/tengchaoyi12/lyric-learning-extension.git
cd lyric-learning-extension
```

2. 安装依赖并编译
```bash
npm install
npm run build
```

3. 加载扩展到 Chrome
   - 打开 `chrome://extensions/`
   - 开启右上角「开发者模式」
   - 点击「加载已解压的扩展程序」
   - 选择项目中的 `dist/` 目录

4. 开始使用
   - 打开 QQ 音乐 Web 版：https://y.qq.com
   - 播放任意歌曲
   - 点击浏览器工具栏的扩展图标
   - 设置重复次数，开始学习

## 快捷键

| Windows/Linux | Mac | 功能 |
|--------------|-----|------|
| `Alt + ←` | `Option + ←` | 手动跳转上一句 |
| `Alt + →` | `Option + →` | 手动跳转下一句 |
| `Alt + R` | `Option + R` | 重置当前句计数 |

> **注意**：在 Mac Chrome 中，Alt 键对应 Option 键。

## 技术架构

```
lyric-learning-extension/
├── manifest.json           # Chrome 扩展配置
├── popup/                  # 弹窗界面
│   ├── popup.html
│   ├── popup.css
│   └── popup.ts
├── content/                # 内容脚本（注入页面）
│   ├── content.ts         # 主入口
│   ├── counter.ts         # 计数逻辑
│   ├── lyric-detector.ts   # 歌词检测
│   └── play-controller.ts  # 播放控制
├── background/             # 后台脚本
│   └── service-worker.ts
└── shared/                 # 共享类型
    └── types.ts
```

- **Manifest V3**：使用最新的 Chrome 扩展规范
- **TypeScript**：类型安全
- **纯 DOM 监听**：不拦截网络请求，依赖 QQ 音乐页面 DOM 结构

## 常见问题

**Q: 扩展无法激活？**
请确认已在 `chrome://extensions/` 中正确加载扩展，且处于启用状态。

**Q: 歌词检测不工作？**
QQ 音乐 Web 版可能更新了页面结构，歌词检测功能可能暂时失效。请提交 Issue 反馈。

**Q: 如何调整重复次数？**
点击扩展图标，在弹窗中点击 `+` 或 `-` 按钮调整，设置即时生效。
